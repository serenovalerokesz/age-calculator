# Age Calculator

A simple browser-based age calculator.  
Enter your date of birth and the app shows your exact age in years, months, days, hours, minutes and seconds.

The seconds update live so your age keeps counting in real time.

## Features

- Calculates exact age from date of birth
- Displays years, months, days, hours, minutes and seconds
- Seconds update live every second
- Multiple language support
- Simple and responsive interface
- Runs entirely in the browser

Supported languages include English, Turkish, Spanish, German, French, Italian, Portuguese, Russian, Arabic, Chinese, Japanese, Korean and Hindi.

## Demo

To run the project locally:

1. Download or clone the repository
2. Open the project folder
3. Double-click `index.html`

The application will open directly in your browser.

## Usage

1. Select your **date of birth**
2. Click **Calculate Age**
3. Your age will be displayed instantly
4. The seconds counter will continue updating automatically

You can change the interface language using the dropdown menu.
